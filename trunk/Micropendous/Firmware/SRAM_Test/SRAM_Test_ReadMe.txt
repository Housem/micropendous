Purpose: Enumerates as a Serial Port (COM Port) with
	 access to the external SRAM on a Micropendous board.

	 After connecting a board programmed with this firmware
	 to your PC, start a serial terminal (such as PuTTY or
	 GTKterm) and connect to the board.  Type h to list
	 the possible SRAM access commands.

	 TestExternalSRAM.py will test all external SRAM with
	 pattern as well as random data.  Run the script in
	 a Terminal or Command Prompt to see usage instructions.
	 Requires Python2 v2.5+ and PySerial
